# Security Policy

## Supported versions

The project is pre-1.0. Security fixes are applied to the latest release and
`main`.

## Reporting a vulnerability

Please report security issues privately via GitHub's
[private vulnerability reporting](https://github.com/zkrebbekx/go-mermaid/security/advisories/new)
rather than opening a public issue.

You can expect an acknowledgement within a few days. Since this library parses
untrusted input (diagram source), reports about parser resource exhaustion,
panics, or unbounded output are in scope.
